Menu

Navigating the menus is inituitive, just click on the buttons. Additionally, there are keybindings to the buttons. Generally, the key bindings are as follows:

CTRL + [Initial of button text]

In Game

There are a few keys that control the game.

Key(s) Function
A, Left arrow key Move left
W, up arrow key, spacebar Jump
D, Right arrow key Move right
Esc Pause
